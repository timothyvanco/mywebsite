Autonomous robot

VISION

  • The main goal of this project was to gain new knowledge in the field of robotics, technical skills in design and production of printed circuit boards and to improve programming skills. This is the reason why I decided to make a robot operated autonomously through sonars. Subsequently I also added a camera module, which monitors field in front of the robot and also sends images via Wi-Fi. I wrote the code for object colour and human skin recognition. I also added a code for robot speech, so now Vision can say something about itself. Newest upgrade brings system, which can measure temperature, humidity and luminance in Vision's proximity.

Smart home

JARVIS

  • Jarvis is a system that can measure, show and control home by my voice commands. This intelligent home consists of IoT components that communicate wirelessly with the control unit with MQTT protocol. It consists of 9 IoT modules.
  • MEASURE Temperature, illumination, humidity, pressure, concentration of gases (including flammable) and determine air quality, soil moisture, detect human presence, open / close door or window.
  • CONTROL Open / close the garage door via RF communication. Turn on the light in the room or illuminate in any color with the LED strip. Full control of TV and radio. PSD regulator to control light and temperature with the user setpoint values.
  • SYSTEM Measured data and signal strength of individual modules are displayed. Jarvis also has intelligence which can control house. It can be triggered by time, by high CO2 or flammable gas values, or person in the room detected and many more...
